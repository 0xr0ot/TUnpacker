# TUnpacker
  
          
## 简介
###*TUnpacker*是一款Android脱壳工具
######*TUnpacker* is an Android unpack tool.
##使用方法
	python tunpacker.py jiagu.apk
##必读事项
	1.本代码仅适用于特定的加固方式 (请参考脚本内容)
	2.本代码仅供安全研究及授权测试使用，如用于非法用途，后果自负
	3.运行本代码前需要确保连接Android测试设备或虚拟机，并确保Android系统已root
